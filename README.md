# Bash-утилита для работы с пользователями и процессами

## Консольная утилита на языке Bash для отображения информации о пользователях и процессах в системе с поддержкой логирования и обработки ошибок.

## Функциональность
-  Вывод списка пользователей и их домашних директорий
-  Вывод списка запущенных процессов
-  Перенаправление вывода в файл
-  Перенаправление ошибок в файл
-  Проверка прав доступа к файлам
-  Поддержка коротких и длинных опций

# Сделать исполняемым
chmod +x bash_util.sh

## Использование
```bash
# Показать справку
./bash_util.sh -h
./bash_util.sh --help

# Вывести пользователей
./bash_util.sh -u
./bash_util.sh --users

# Вывести процессы
./bash_util.sh -p
./bash_util.sh --processes

# Сохранить вывод в файл
./bash_util.sh -u -l /tmp/users.txt
./bash_util.sh --users --log /var/log/output.log

# Сохранить ошибки в файл
./bash_util.sh -p -e /tmp/errors.log
./bash_util.sh --processes --errors /var/log/errors.log

# Комбинированное использование
./bash_util.sh -u -p -l output.txt -e errors.log
